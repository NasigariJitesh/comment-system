## Backend
In the project directory,
### `cd backend`

 create a .env file and add the db access password
 
To start Server
### `yarn start`


## Frontend
In the project directory,
### `cd frontend`

To start Server
### `yarn start`
Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.


SignUp as a new User and then login 
After logging in on Home page you can view and upload the documents

To edit or comment on a ducument click on the particular document

To comment any text in the document select the text and right click to display the menu, then choose comment
